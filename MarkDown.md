# MarkDown 문법

* #, ##, ###, ... ... 은 html의 `<h1>`, `<h2>`, `<h3>`와 같음. 
* 고유명사는 `*`으로 감싸서 이탤릭체로 씀.
* 볼드체는 `**`로 감싸면 됨.



## 리스트

* 문장처음에 `*`쓰고 띄어쓰기하면 `<ul>`처럼 됨
* 문장처음에 `1.`쓰고 띄어쓰기하면 `<ol>`처럼 됨



## 코드

* 코드 내용을 쓸 땐`'`으로 감싸줌.(`'`를 *backkick*이라 함.)

* 긴 코드 쓸때  ````프로그램언어이름` 쓰면  창 뜸
